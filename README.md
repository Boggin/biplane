# biplane

Control your Kong instance through simple configuration.

[biplane](http://drops.articulate.com/w1ad/NQlLBhGp+)

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  biplane:
    github: plukevdh/biplane
```
